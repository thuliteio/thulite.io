Bolt is a minimal and elegant blog theme for Thulite, designed to provide a fast, distraction-free reading experience. With its lightweight footprint and clean styling for all HTML elements, Bolt is ideal for writers and bloggers who value simplicity and performance. The theme includes thoughtful features such as dark/light mode switching, tag support, reading time estimates, and related posts to enhance user engagement.

## Features

- Minimal and responsive layout
- Consistent HTML elements styling
- Dark mode and light mode toggle
- Tag support for organizing content
- Estimated reading time display
- Related posts suggestions for better navigation

## Credits

The HTML elements styling is based on [Bolt.css](https://boltcss.com/). The dark mode toggle is based on [The Complete Guide to the Dark Mode Toggle](https://ryanfeigenbaum.com/dark-mode/).
